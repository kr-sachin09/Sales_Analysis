This project analyzes sales data to generate actionable insights for business decision-making.
The dataset includes orders, customers, product categories, sales, profit, and discounts.
Key KPIs such as total sales, profit, margin, orders, and customers were calculated.
Category-wise analysis identified Technology as the most profitable segment.
Office Supplies showed stable performance with healthy margins.
Furniture recorded high sales but low profit, highlighting improvement opportunities.
A professional executive summary PDF was created for management use.
Tools used include Python (Pandas) and ReportLab for PDF reporting.
